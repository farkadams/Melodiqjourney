# Az ÁSZF célja

Jelen Általános Szerződési Feltételek (a továbbiakban: **ÁSZF**) szabályozza a Melodiq Journey weboldalán és kapcsolódó szolgáltatásain keresztül elérhető szolgáltatások igénybevételének feltételeit.

Az ÁSZF meghatározza különösen:

- a weboldal használatának szabályait;
- a felhasználói jogokat és kötelezettségeket;
- az eseményekkel kapcsolatos rendelkezéseket;
- a buszos utazások részvételi feltételeit;
- a Journey Club tagság szabályait;
- a partnerkedvezmények igénybevételének feltételeit;
- a jövőben bevezetésre kerülő online szolgáltatások alapvető szabályait;
- a felelősségi szabályokat;
- a jogviták rendezésének módját.

Az ÁSZF minden olyan természetes személyre és szervezetre vonatkozik, aki a Melodiq Journey weboldalát használja vagy annak bármely szolgáltatását igénybe veszi.
